Red Ink Excel Helper
===================

1. Ensure that no Red Ink helper is loaded by Excel.
2. Open Excel
3. Open a new worksheet
4. Go the Visual Basic Project Explorer
5. Rename the project of your worksheet to "redink_helper"
6. Import "RI_Wait" form
7. Import the "RI_Helper_Code_Excel.bas" file
8. Set your digital signature (optional)
9. Save the file as a Excel addin under the name "redink_helper.xlam"
10. Copy the helper to the Excel xlstart directory
