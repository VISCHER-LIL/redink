Red Ink Word Helper
===================

1. Ensure that no Red Ink helper is loaded by Word.
2. Open redink_cleanempty.dotm
3. Go the Visual Basic Project Explorer
4. Rename the project of "redink_cleanempty.dotm" to "redink_helper"
5. Import the .bas file
6. Set your digital signature (optional)
7. Save the document as dotm file under the name "redink_helper.dotm"
8. Copy the helper to the Word startup directory
